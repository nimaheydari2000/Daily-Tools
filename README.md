🛠️ Daily Tools
A collection of small Python utilities for everyday tasks.

Tools
📄 PDF to Markdown
Converts PDF files into clean Markdown format.
Usage:
bashpython pdf_to_markdown.py input.pdf
Output:
A .md file with the converted content, saved in the same directory.
Requirements:
bashpip install -r requirements.txt

Getting Started

Clone the repo:

bash   git clone https://github.com/your-username/daily-tools.git
   cd daily-tools

Install dependencies:

bash   pip install -r requirements.txt

Run any tool (see usage above)


Requirements

Python 3.8+
